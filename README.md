# Frontend - Progressive Web App

Aplicación frontend desarrollada como **Progressive Web App (PWA)**, preparada para ofrecer una experiencia similar a una app nativa en dispositivos web, incluyendo soporte para **notificaciones push**.

## Descripción

Este proyecto corresponde a la capa frontend de la aplicación. Su objetivo principal es permitir la interacción del usuario desde una interfaz web moderna, responsive e instalable, incorporando funcionalidades propias de una PWA como:

- Instalación en dispositivos compatibles
- Funcionamiento optimizado en entorno web móvil
- Soporte para notificaciones push
- Experiencia de usuario fluida y accesible

## Funcionalidades principales

- Interfaz de usuario para la aplicación
- Registro y manejo de notificaciones push
- Configuración como Progressive Web App
- Integración con servicios/backend para el envío y recepción de datos

## Tecnologías utilizadas

- Framework/library frontend: React | JWT
- JavaScript / TypeScript
- HTML / CSS
- Service Workers
- Web Push Notifications

## Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Node.js
- npm o yarn

## Instalación

-Clonar el repositorio.
-npm install.
-Levantar proyecto con npm run dev.
